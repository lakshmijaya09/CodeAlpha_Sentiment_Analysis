Sentiment Analysis on Customer Reviews

## Project Overview
This project performs Sentiment Analysis on customer reviews using Natural Language Processing (NLP) techniques. The system analyzes textual reviews and classifies them into Positive, Negative, and Neutral sentiments.

The project helps understand customer opinions, product feedback, and public sentiment trends using Python.

---

## Objectives
- Analyze customer review data
- Classify sentiments as Positive, Negative, or Neutral
- Understand customer opinions and emotions
- Visualize sentiment patterns using graphs
- Support marketing and business insights

---

## Dataset Information
The dataset contains sample customer reviews collected from online platforms such as Amazon reviews, social media comments, and product feedback systems.

### Sample Reviews
- Product is amazing
- Very bad quality
- I love this item
- Worst experience ever
